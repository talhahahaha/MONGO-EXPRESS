# Chat App (Express + MongoDB)

A simple chat application built using **Express**, **MongoDB**, and **EJS**. This project demonstrates CRUD operations (Create, Read, Update, Delete) on chat messages.

## 🛠️ Features

- View all chat messages
- Add a new chat
- Edit existing messages
- Delete messages
- Server-side rendering using EJS
- Method override for PUT and DELETE support in HTML forms

## 📁 Project Structure


## ⚙️ Technologies Used

- Node.js
- Express.js
- MongoDB (Mongoose)
- EJS
- Method-Override
- Body-parser (via `express.urlencoded`)
- HTML AND CSS

## 🚀 Getting Started

### Prerequisites

- Node.js and npm installed
- MongoDB running locally (on `mongodb://127.0.0.1:27017`)
